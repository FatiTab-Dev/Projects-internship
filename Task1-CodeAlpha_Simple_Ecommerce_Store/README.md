# Simple E-commerce Store

A functional e-commerce web application featuring product listings, a shopping cart, and secure order processing.

## Key Features

- **Product Management:** Browse products with detailed views.
- **Shopping Cart:** Efficient state management for adding, removing, and updating items.
- **Secure Authentication:** User registration and login protected with **JWT**.
- **Data Security:** User passwords are encrypted using **bcrypt** to ensure data protection.
- **API Integration:** Seamless communication between frontend and backend using **Axios**.

## Tech Stack

- **Frontend:** React.js, Bootstrap, CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
