# Collaborative Project Management Tool

A Trello-inspired collaborative tool designed for teams to manage tasks and projects efficiently.

## Key Features

- **Project Boards:** Group-based project organization.
- **Drag & Drop:** Intuitive task management using `@hello-pangea/dnd`.
- **Real-time Collaboration:** Instant notifications and live updates integrated with **Socket.io**.
- **Secure Auth:** JWT-based authentication combined with **bcrypt** for password security.
- **Communication:** Interactive comments within task cards.

## Tech Stack

- **Frontend:** React.js, Bootstrap, CSS, @hello-pangea/dnd
- **Backend:** Node.js, Express.js, Socket.io
- **Database:** MongoDB
